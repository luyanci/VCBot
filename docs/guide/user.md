## 禁言play

### 自裁

可以禁言自己随机时长/自定义时长

```
自裁 [禁言时间(s)]
```

### 决斗

需要两位用户发送`决斗`，会随机决斗，决斗成功则禁言对方，失败则禁言自己（时长随机）

### 21点

不限人数，可自由参加，具体使用方法可发送 `21点 帮助` 查看`

### 轮盘

不限人数，可自由参加，玩法参考恶魔轮盘赌
