## bili
b站推送命令（目前仅列举常用部分）
### bili add {uid} 
b站添加订阅命令，添加后，检测到更新时会推送到设置的群聊中

### bili config {uid} [推送群号]
b站推送配置命令，可设置at全体推送/过滤器/设置推送主题色等
